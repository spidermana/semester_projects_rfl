      Structure of the Rust and C Binderfs.

      \texttt{inode->i\_fop}, so the VFS dispatches to either implementation identically.
      The Rust driver decomposes the monolithic \texttt{binder.c} into typed modules
      and draws on \texttt{workqueue.rs} as a safe abstraction over the C workqueue.}
